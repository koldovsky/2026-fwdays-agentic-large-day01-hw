Цей проєкт є форком або локальною копією **Excalidraw** — популярного open-source інструменту для створення діаграм, схем та ескізів із візуальним стилем "намальовано від руки" (hand-drawn).

### Призначення
Основне завдання проєкту — надання віртуальної дошки (whiteboard) для колаборації, малювання інтерфейсів, архітектурних схем чи будь-яких інших візуалізацій. Він може працювати як самостійний вебдодаток, так і як NPM-бібліотека для вбудовування в інші проєкти.

### Основний стек технологій
* **Мови програмування:** TypeScript, JavaScript.
* **Бібліотеки інтерфейсу:** React (з використанням хуків та функціональних компонентів).
* **Управління станом:** Jotai.
* **Інструмент збірки та розробки:** Vite.
* **Тестування:** Vitest (з jsdom).
* **Зберігання даних та колаборація:** Firebase ( Firestore, Storage).
* **Стилізація:** SCSS (Sass).
* **Менеджери пакетів:** Yarn / NPM.

### Структура репозиторію
Проєкт побудований як монорепозиторій і складається з наступних ключових директорій:

* **`excalidraw-app/`** — безпосередньо вебдодаток (застосунок), який компілюється як PWA і зазвичай розгортається для кінцевих користувачів. Містить логіку ініціалізації, колаборації (через Firebase) та інтеграції зовнішніх сервісів.
* **`packages/`** — набір незалежних NPM-пакетів:
  * `excalidraw` — основна абстракція редактора та React-компоненти (використовується для встановлення через `npm install @excalidraw/excalidraw`).
  * `common` — спільні утиліти, константи, кольори.
  * `math` — математичні рушії, типи (наприклад `Point`) та функції для розрахунків геометрії полотна.
  * `utils` — утиліти для експорту малюнків у JSON, SVG, зображення (`exportToBlob`, `exportToSvg`).
  * `element` — типи та логіка керування окремими елементами полотна (лінії, прямокутники, текст).
* **`examples/`** — приклади інтеграції бібліотеки Excalidraw з іншими технологіями, наприклад з Next.js (`with-nextjs`) або класичними браузерними скриптами (`with-script-in-browser`).
* **`firebase-project/`** — конфігураційні файли, правила безпеки та індекси для Firebase.
* **`scripts/`** — допоміжні NodeJS скрипти для процесів збірки, автоматизації релізів, генерації WASM шрифтів, збірки локалізацій тощо.
* **`public/`** — статичні файли, іконки, PWA manifest (`service-worker.js`), маніфести та кастомні шрифти (наприклад Virgil та Cascadia).
