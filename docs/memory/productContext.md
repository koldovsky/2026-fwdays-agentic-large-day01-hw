# Product Context

## UX-цілі

### Головна ціль

Надати **максимально простий** інструмент для швидкого скетчингу діаграм із відчуттям малювання від руки — без кривої навчання, без реєстрації, одразу в браузері.

### Ключові UX-принципи

- **Zero friction** — відкрив сторінку, одразу малюєш (без signup)
- **Handdrawn aesthetic** — Rough.js + Perfect-freehand дають відчуття "нічого не фінального", що знижує психологічний бар'єр для ідей
- **Collaboration-first** — поділитися посиланням = 1 клік, шифрування "з коробки"
- **Portable** — PWA для офлайн-режиму, експорт у PNG/SVG/JSON
- **Embeddable** — бібліотека `@excalidraw/excalidraw` для вставки в сторонні додатки

## Аудиторія та сценарії

### Аудиторія 1 — Кінцеві користувачі (excalidraw.com)

| Сценарій | Дія | Очікуваний результат |
|----------|-----|----------------------|
| Швидка ідея | Відкрити сайт → намалювати схему | ≤30 сек до першого штриха |
| Презентація | Намалювати → поділитися PNG | Чистий PNG зі скетч-стилем |
| Колаборація | Надати посилання колезі | Реальний час, без лагу |
| Офлайн | Немає інтернету | Застосунок відкривається з PWA |
| Мобільний | Touch-drawing | Плавні лінії через Perfect-freehand |

### Аудиторія 2 — Розробники (npm-пакет)

| Сценарій | Дія | Очікуваний результат |
|----------|-----|----------------------|
| Вбудувати whiteboard | `npm install @excalidraw/excalidraw` | React-компонент, готовий з props |
| Завантажити дані | `initialData` prop | Відновити збережену сцену |
| Отримати зміни | `onChange` callback | Отримати актуальний стан сцени |
| Кастомізувати UI | `renderTopRightUI` | Додати власні кнопки в тулбар |
| Next.js SSR | Dynamic import + `ssr: false` | Компонент без SSR-помилок |

### Аудиторія 3 — Self-hosters

| Сценарій | Дія | Очікуваний результат |
|----------|-----|----------------------|
| Docker deploy | `docker-compose up` | Повний стек локально |
| Firebase налаштування | Власний Firebase project | Колаборація на своїй інфраструктурі |

## Ключові User Flows

### Flow 1: Одиночне малювання

```text
Відкрити URL → Вибрати інструмент (панель зліва) → Намалювати
→ Виділити → Змінити стиль (права панель) → Експортувати
```

### Flow 2: Колаборація

```text
Намалювати → Live collaboration (іконка в меню) → Поділитися посиланням
→ Колега відкриває URL → Бачить малюнок + курсори → Спільне редагування
```

### Flow 3: Вбудування в застосунок

```text
npm install → import Excalidraw → Передати initialData →
Підписатися на onChange → Зберігати стан → Відновити при перезавантаженні
```

## Пріоритети UX

1. **Швидкість першого малювання** — жодних модальних вікон при старті
2. **Колаборація** — посилання з ключем шифрування в URL-хеші
3. **Accessibility** — Radix UI компоненти, ARIA-атрибути
4. **Мультимовність** — 80+ локалей через i18next, lazy-load чанки
5. **Продуктивність** — два canvas (interactive + static), lazy-load шрифтів

---

## Обмеження UX (поточні)

- Складні форматування тексту не підтримуються (plain text в елементах)
- Анімації/відео — поза скоупом
- Версійна історія — тільки undo/redo в межах сесії (не cloud history)
- Мобільний UX — обмежений (touch підтримується, але не оптимізований)
