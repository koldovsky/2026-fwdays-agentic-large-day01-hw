# Product context — Excalidraw (цей форк)

## Навіщо продукт існує

Користувачу потрібно **швидко** накреслити ідею, схему або пояснення **без** важкого дизайн-інструменту — у браузері, з можливістю поділитися або вбудувати редактор у свій сайт.

## Хто користувачі

- **Автори діаграм**: нотатки, скетчі процесів, проста візуалізація.
- **Команди**: спільне редагування (коли увімкнена колаборація та бекенд).
- **Розробники**: вбудовування через `@excalidraw/excalidraw` у React-застосунки.

## Типові сценарії

- Намалювати блок-схему / стрілки / фігури / текст.
- Експортувати PNG/SVG або зберегти `.excalidraw` / поділитися посиланням (залежить від деплою).
- Імпорт діаграм з Mermaid у елементи сцени (фіча продукту в апстрімі).
- Офлайн / PWA — базова робота без мережі після завантаження.

## Продуктові обмеження (коротко)

- Не **CAD** і не **професійний векторний редактор** — фокус на швидкості та простоті.
- Зберігання сцени: **локально**, **експорт файлом**, через **інтеграції** (`excalidraw-app`: Firebase, collaboration) — один репозиторій не дорівнює одній моделі «хмари».
