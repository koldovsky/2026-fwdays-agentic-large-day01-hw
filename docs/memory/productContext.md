# Product Context

## Product intent
- Excalidraw is a browser-based whiteboard for sketching diagrams and visual thinking.
- It supports quick drafting, editing, and sharing of scenes.

## Primary user intents
- Draw and edit elements on a canvas.
- Organize/transform visual content quickly.
- Share scenes through links or collaboration sessions.
- Export work to files/images and re-import later.

## Experience qualities
- Low-friction first use (`<Excalidraw />` embed model and app shell UX).
- Real-time collaboration support.
- Internationalization support through locale bundles.
- Feature depth without requiring heavy setup for basic drawing.

## Product boundaries
- Core value is interactive canvas editing and scene portability.
- Not a full PM/document system; product focuses on visual ideation and diagram workflows.

## Current doc gaps to watch
- Some implicit runtime behaviors (sync timing, export waiting, scene initialization branches) are code-driven and not fully surfaced in user-facing docs.
