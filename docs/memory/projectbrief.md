# Project Brief: Excalidraw

## Overview

Excalidraw is an open-source (MIT License) collaborative virtual whiteboard application with a hand-drawn, sketch-like aesthetic. It serves both as a standalone web app at excalidraw.com and as an embeddable React component (`@excalidraw/excalidraw`) for third-party integrations.

## Core Mission

Provide a simple, intuitive, and collaborative diagramming tool that makes technical diagrams, wireframes, and brainstorming feel natural — like sketching on paper, but with digital superpowers.

## Key Goals

1. **Simplicity** — zero learning curve, no accounts required to start drawing
2. **Collaboration** — real-time multi-user editing via WebSocket
3. **Portability** — embeddable React component for any web application
4. **Performance** — smooth canvas rendering even with hundreds of elements
5. **Openness** — MIT licensed, community-driven development
