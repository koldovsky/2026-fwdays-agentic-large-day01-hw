# Project brief — Excalidraw (monorepo)

## Суть проєкту

Це **monorepo на базі Excalidraw** — відкритого інструменту для малювання діаграм і скетчів у стилі «доски» в браузері. Користувачі створюють сцени з примітивів (фігури, лінії, стрілки, текст, зображення тощо), експортують і діляться ними.

## Основна мета

- Дати **швидкий візуальний whiteboard** з мінімальним фрикційним UX.
- Забезпечити **вбудованість**: пакет `@excalidraw/excalidraw` як React-компонент для сторонніх застосунків.
- Підтримати **реальну співпрацю** та збереження сцен (у повному застосунку — через `excalidraw-app`: Firebase, socket.io тощо залежно від конфігурації).

## Що входить у цей репозиторій (рівень репо)

| Частина | Призначення |
| --- | --- |
| `packages/excalidraw` | Ядро редактора: UI, canvas, дії, стан |
| `packages/element` | Модель і логіка елементів сцени |
| `packages/common` | Константи, утиліти, спільні типи |
| `packages/math` | Геометрія та математика для редактора |
| `packages/utils` | Додаткові утиліти пакету |
| `excalidraw-app` | Prod-застосунок (Vite), інтеграції (Jotai, Firebase, Sentry) |
| `examples/*` | Приклади вбудовування |

## Не є ціллю цього опису

Детальна історія продукту upstream, roadmap комерційних сервісів і повний перелік інтеграцій — задокументовані в офіційному репозиторії Excalidraw; тут — контекст **цього** форку/worktree для агентів і розробників.
