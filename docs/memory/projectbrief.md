# Project brief — Excalidraw monorepo

## Що це за проєкт

**Excalidraw** — відкритий веб-редактор для малювання діаграм і схем у стилі «ручного» скетчу (hand-drawn). Основний продукт — інтерактивна дошка на canvas з підтримкою фігур, стрілок, тексту, фреймів, експорту та (у повному додатку) колаборації.

Цей репозиторій — **monorepo** на базі офіційного Excalidraw: окремий пакет `@excalidraw/excalidraw` (React-компонент для вбудовування) і додаток **`excalidraw-app`** (повноцінний застосунок зі збіркою через Vite).

## Основна мета

- Дати користувачам швидкий інструмент для візуального мислення: скетчі UI, архітектурні діаграми, нотатки на дошці.
- Забезпечити **вбудовування** редактора в сторонні продукти через npm-пакет і чіткий React API.
- Зберігати **відкритий код** і розширюваність (дії, інструменти, експорт, локалізації).

## Для кого

- Індивідуальні користувачі (швидкі схеми, нотатки).
- Команди (колаборація, спільні дошки в hosted-додатку).
- Розробники, які вбудовують canvas-редактор у свої застосунки.

## Ключові можливості (рівень продукту)

- Малювання й редагування векторних елементів на canvas.
- Інструменти виділення, групування, прив’язки стрілок до елементів.
- Теми, сітка, zoom/pan, zen/view режими.
- Експорт зображення та JSON сцени; бібліотека фігур.
- У monorepo: окремі пакети для математики, моделі елементів, спільних констант.

## Межі проєкту (коротко)

- Це не повноцінний графічний редактор типу Figma/Illustrator; фокус — швидкі діаграми та дошка.
- Hosted-колаборація та облікові інтеграції реалізовані на рівні **`excalidraw-app`** і зовнішніх сервісів, а не в мінімальному embed-only сценарії.

## Структура репозиторію (верхній рівень)

| Шлях | Призначення |
|------|-------------|
| `excalidraw-app/` | Продуктовий застосунок (Vite, React 19) |
| `packages/excalidraw/` | Головний пакет редактора (UI, рендер, дії) |
| `packages/element/` | Типи та операції над елементами |
| `packages/math/` | Геометрія та точки |
| `packages/common/` | Константи, спільні типи та утиліти |
| `packages/utils/` | Допоміжні утиліти |
| `examples/` | Приклади інтеграції |
