# Product Context

This file is a compact product navigation layer for Memory Bank.
Detailed product behavior is documented in `docs/product/*`.

## Product Intent (Compact)
- Provide a fast browser whiteboard/diagramming experience for both standalone users and integrators.
- Keep local-first editing as baseline, with collaboration/share/library/AI as additive capabilities.
- Preserve backward compatibility for scenes and libraries while evolving features.

## Primary Audiences
- Developers embedding `<Excalidraw />` in React apps.
- End users drawing, importing/exporting, and collaborating.
- Maintainers extending runtime behavior while preserving compatibility.
