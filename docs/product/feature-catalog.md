# Feature Catalog

## Core Editor
- Canvas-based diagram editing via `<Excalidraw />`.
- Tooling and UI flows via `LayerUI`, menu, toolbar, dialogs.

## Collaboration and Sharing
- Collaboration integration in app shell (`excalidraw-app/collab`).
- Import/export and scene serialization through data modules.

## Embedding and Integration
- Published package `@excalidraw/excalidraw` for React embedding.
- Example integrations: script-in-browser and Next.js.

## Platform Features
- Undo/redo through store+history pipeline.
- Export support (PNG/SVG/canvas helpers).
