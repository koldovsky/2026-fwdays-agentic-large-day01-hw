# PRD (reverse-engineered) — Excalidraw

## Product purpose

Excalidraw — це **веб-застосунок для малювання діаграм і скетчів** у стилі рукописного whiteboard: швидке створення фігур, стрілок, тексту та схем з можливістю експорту та спільної роботи. Продукт також постачається як **React-компонент** для вбудовування в інші сайти.

## Target audience

- **Індивідуальні користувачи**: скетчі, пояснення, проста візуалізація ідей.
- **Команди**: спільне редагування діаграм у браузері.
- **Розробники**: інтеграція редактора через пакет `@excalidraw/excalidraw`.

## Key features

1. **Інструменти малювання**: прямокутники, еліпси, лінії, стрілки, фрі-хенд, текст, зображення, фрейми тощо.
2. **Редагування сцени**: виділення, групування, шари (порядок елементів), прив’язка стрілок до об’єктів (binding).
3. **Експорт**: PNG/SVG та інші режими залежно від UI; збереження сцени у форматі, придатному для відкриття в Excalidraw.
4. **Бібліотека фігур**: повторне використання наборів елементів (library).
5. **Колаборація** (у повному застосунку): синхронізація сцени між клієнтами, список учасників.
6. **Локалізація**: переклади інтерфейсу (файли локалей у `packages/excalidraw/locales/`).
7. **Mermaid / діаграми**: імпорт/перетворення в елементи сцени (залежності `@excalidraw/mermaid-to-excalidraw`).
8. **PWA**: офлайн/встановлення як застосунок (Vite PWA у кореневій збірці).
9. **Доступність і форм-фактори**: адаптація під мобільні пристрої (окремі тести/UI-гілки).

## Technical constraints & non-goals (орієнтовні)

- **Не** є повноцінним векторним редактором рівня Illustrator/Inkscape (фокус на швидких діаграмах, не на друкарській верстці).
- **Не** гарантує ідентичність рендерингу в усіх браузерах на рівні субпікселя — орієнтир на сучасні браузери (`browserslist` у пакетах).
- Продуктовий застосунок покладається на **мережеві сервіси** (Firebase, collaboration server) для повної колаборації; локальний пакет може працювати без них з обмеженим функціоналом.
